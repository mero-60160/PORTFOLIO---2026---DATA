## 🛠️ Stack Technique

```
📦 portfolio-la-manu
├── 📄 index.html          # Page principale (SPA)
├── 📁 assets/
│   ├── 📁 css/
│   │   └── 🎨 style.css   # Design system complet
│   ├── 📁 js/
│   │   └── ⚡ script.js   # Moteur d'animations
│   └── 📁 img/
│       └── 🖼️ profile.png # Photo de profil
└── 📄 README.md           # Ce fichier
```

- **HTML5** sémantique avec SEO optimisé (meta tags, Open Graph)
- **CSS3** avec variables custom, animations keyframes, glassmorphism
- **JavaScript** vanilla — Canvas API, IntersectionObserver, requestAnimationFrame
- **Google Fonts** — Inter & JetBrains Mono
- **Font Awesome 6** — Icônes vectorielles
